**캘린더 셀 표시 규칙:**
- 일정 바(Bar): 최대 **3개** 표시. 초과 시 `+N` 텍스트.
- 지출 합계: 해당 날짜의 총 지출을 `-#,###` 형식으로 표시.
- 다일 일정: 시작~종료일에 걸쳐 연속 바로 렌더링.

**하단 리스트 영역:**
- 선택된 날짜가 없을 때 → **오늘**의 내역을 기본 표시.
- 일정과 가계부를 **섹션 분리**하여 표시.
- 각 항목에 `[Personal]` 또는 `[Joint]` 뱃지 표시.

**필터 동작:**

| 상태 | All 필터 | Me 필터 | Joint 필터 |
|------|---------|---------|-----------|
| 싱글(PERSONAL 그룹) | 전체 표시 | All과 동일 | 숨김 |
| 공유(SHARED 그룹) | 전체 표시 | 내 Personal만 | Joint만 |

- 싱글 모드(PERSONAL 그룹): Type 필드 숨김. 자동 Personal.
- 공유 모드(SHARED 그룹): `[Personal]` `[Joint]` 버튼 노출. 기본값 Personal.
- Joint 선택 시: 모달 테마 색상이 해당 그룹의 `joint_asset_color`로 변경.

**일정 연결 (Link to Schedule):**
- Expense/Income 탭에서만 노출.
- 선택된 날짜의 일정 목록을 드롭다운으로 표시.
- 일정이 없으면 "등록된 일정이 없습니다" 표시.

**수정 모드 추가 요소:**
- 상단에 **[삭제]** 버튼 표시 (빨간색 텍스트).
- 일정 삭제 시: "연결된 가계부 N건의 연결이 해제됩니다" 확인 팝업.
